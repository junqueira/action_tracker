How to Contribute
=================

Issues and bugs
---------------
    To create issues or report bugs. Plase use Github :D

    https://github.com/valdergallo/django-choices-flow/issues


Create my SandBox
------------------

.. code-block:: bash
    :linenos:

    # download code
    git clone git@github.com:valdergallo/django-choices-flow.git

    # install developer packages
    make

    # check coverage
    make coverage

    # test project
    make test

    #clean extra content
    make clean

    #send package
    make send_package

    #test py2 and py3
    tox
