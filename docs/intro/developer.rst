How to Contribute
=================

Issues and bugs
---------------
    To create issues or report bugs. Plase use Github :D

    https://github.com/nokiadev/action_tracker/issues


Create my SandBox
------------------

.. code-block:: bash
    :linenos:

    # download code
    git clone https://github.com/nokiadev/action_tracker.git

    # install developer packages
    make dev

    # test project
    make test

    #clean extra content
    make clean
